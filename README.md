# Sunny Systems — Brand Identity V4 (2026)

Repositorio da nova identidade visual e marca da Sunny Systems. Contem a historia da evolucao da marca, documentacao de brand guidelines, design tokens, assets visuais e propostas interativas.

## Estrutura

```
assets/
  v1/          Logos e manual da identidade original (2021)
  v2/          Logos e paleta do Branding 2.0 (2023)
docs/
  historia-da-marca.md          Evolucao completa: V1 → V2 → V3 → V4
  proposta-a-convergence.html   Proposta A — Convergence
  proposta-a2-aurora.html       Proposta A2 — Aurora (variante)
  proposta-b-radiance.html      Proposta B — Radiance
```

## Propostas V4

Tres propostas interativas em HTML self-contained (inline CSS + SVG + Google Fonts). Abra diretamente no browser.

| Proposta | Conceito | Tipografia | Simbolo |
|----------|----------|------------|---------|
| **A — Convergence** | 4 pilares convergindo ao centro | Quicksand + Inter | Pinwheel 6 raios |
| **A2 — Aurora** | Sol nascente, novo amanhecer | Quicksand + Inter | Semicirculo + 5 raios ascendentes |
| **B — Radiance** | Luz que emana da escuridao | Sora + DM Sans | Sol geometrico 8 raios |

Todas compartilham o DNA visual confirmado:

- Dark mode como fundacao
- Amarelo dourado como unica cor accent
- Hierarquia via opacidade de branco
- Glass-morphism e bordas semitransparentes
- Tom tecnico, premium, confiavel
